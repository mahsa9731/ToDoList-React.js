# Modern Interactive Kanban Board

A feature-rich, highly responsive Kanban and Task Management Board built with React, Tailwind CSS, and Framer Motion. Designed with modern UI/UX principles, smooth animations, inline editing, and real-time state tracking.

## Features and Highlights

Why this is not just another standard To-Do List application:

* Modern Minimalist UI: Clean design with vibrant tag badges, custom scrollbars, and intuitive layout.
* Fluid Drag and Drop Interface: Effortlessly move tasks across different columns with visual feedback.
* Inline Task Editing: Quickly update task titles and descriptions directly on the card without modal overlays.
* Subtask Tracking and Animated Progress Bar: Break down complex tasks into subtasks with real-time visual progress indicators.
* Categorized Labels: Color-coded labels (Design, Development, Product, Marketing, etc.) for seamless task categorization.
* Micro-interactions with Framer Motion: Smooth entry/exit animations, hover physics, and dynamic list reordering.
* Search and Filter Capabilities: Easily filter tasks across all status columns in real-time.

## Tech Stack

* Frontend Framework: React 18+
* Build Tool: Vite
* Styling: Tailwind CSS
* Animations: Framer Motion
* Icons: Lucide React
